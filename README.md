# tillless-login

Collection of iOS UIKit UIView classes for Tillless apps

## Installation

Add this line to your application's Gemfile:

    gem 'tillless-restikle'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install tillless-restikle
